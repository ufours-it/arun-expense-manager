# Introduction
Pocket Ledger

Pocket Ledger is a simple and intuitive expense tracking mobile application designed to help users manage daily spending effectively.

It allows users to track expenses with amount, category, date, and notes, along with full support for adding, editing, and deleting entries.

# Get started

Install Dependencies
   npm install

Start the Application
   npx expo start

Run the App
   Scan the QR code using Expo Go (Android / iOS)
   Or run on an Android Emulator / iOS Simulator
